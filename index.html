发表于 2019-04-03 | 分类于 路由器基础设置 | 评论数：

几种模式

模式 标识 命令或说明
用户模式 Router> 开机进入即是；无命令即默认状态
特权模式 Router# enable
全局模式 (config)# congfigure te
接口模式 (config-if)# interface s0/0
路由模式 (config-router)# router rip
访问控制列表模式 (config-std-nacl）# ip access-list standard test

退出命令

退出模式 说明
exit 退到上级命令模式
end 退到特权命令模式

从用户模式到特权模式设置密码的两种方式

方式 命令
文明 enable password cisco
加密 enable secret 1234

备注：show r查看，当同时设置时，加密方式生效，明文方式失效。

hexo搭建

1.搭建前的准备
1.1安装git
1.2安装nodejs
1.3安装hexo
git和nodejs安装提示安装即可，记录下Hexo安装过程中的区别安装
2.Hexo安装过程

新建完成后，指定文件夹目录下有：

  • node_modules: 依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：用来存放你的文章
  • themes：主题
  • _config.yml: 博客的配置文件

打开hexo的服务，在浏览器输入localhost:4000就可以看到你生成的博客了。

参考文档：https://blog.csdn.net/sinat_37781304/article/details/82729029
